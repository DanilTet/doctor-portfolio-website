ІНСТРУКЦІЯ ЩОДО ДОДАВАННЯ ФОТОГРАФІЙ СЕРТИФІКАТІВ
==============================================

Для того щоб фотографії сертифікатів правильно відображалися на сайті або були доступні в модальних вікнах, завантажте ваші файли у папку:
`img/certificates/`

Перейменуйте ваші файли відповідно до списку нижче:

1. Olympus Continuum №04/06:
   -> Назва файлу: `cert-olympus-04-06.jpg` (або .png)

2. Ендо-24:
   -> Назва файлу: `cert-endo24.jpg` (або .png)

3. Endoscopy Days Lviv:
   -> Назва файлу: `cert-lviv-2023.jpg` (або .png)

4. Одеса Ендо 2024:
   -> Назва файлу: `cert-odesa-2024.jpg` (або .png)

5. Інтервенції під контролем УЗ:
   -> Назва файлу: `cert-uz-interventions.jpg` (або .png)

6. НМУ Ендоскопічні інтервенції:
   -> Назва файлу: `cert-nmu-interventions.jpg` (або .png)

7. Запальні захворювання кишечника:
   -> Назва файлу: `cert-ibd-2022.jpg` (або .png)

8. Olympus Continuum №01/11 (Endoscopic Ultrasound):
